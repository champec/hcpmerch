# HCPMERCH - Shopify Theme

A custom Shopify theme designed for **HCPMERCH**, a Print-on-Demand business offering trendy mugs, t-shirts, and accessories for UK Healthcare Professionals.

HCPMERCH provides healthcare professionals in the UK with:

- **Accessories** and other POD items
- **Print-on-Demand** powered by Gelato for fast UK delivery

## üöÄ Installation

1. **Zip the theme folder** (exclude the `.git` directory)
2. **Upload to Shopify**:
   - Go to **Online Store ‚Üí Themes** in your Shopify admin
   - Click **"Upload theme"**
   - Select the zip file
3. **Customize** using the Theme Editor
4. **Publish** when ready
